Under the Hood

1

Signal Generation

The bot subscribes to multiple exchange WebSocket streams to aggregate order book depth. It calculates indicators like RSI and MACD in real-time using Pandas and TA-Lib.

2

Order Execution

Once a signal is confirmed, orders are routed through the CCXT library, which normalizes API calls across 100+ exchanges (Binance, Coinbase, Kraken).

3

Post-Trade Analysis

Every trade is logged to a PostgreSQL database. A separate microservice analyzes trade performance and adjusts strategy parameters dynamically.

class GridStrategy(Strategy):
    def __init__(self, exchange, symbol):
        self.exchange = exchange
        self.grids = []

    async def on_tick(self, ticker):
        current_price = ticker['last']
        
        # Check for buy signals
        if current_price < self.lower_band:
            await self.place_limit_order(
                side='buy', 
                amount=self.position_size
            )
            print(f"Executing BUY at {current_price}")
            
        # Dynamic rebalancing logic
        if self.volatility > 0.05:
            self.adjust_grid_spacing()
